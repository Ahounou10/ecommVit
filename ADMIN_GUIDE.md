# 📱 Guide du Tableau de Bord Admin

## Accès

1. Allez sur `http://localhost:3000/admin/login`
2. Connectez-vous avec votre email Supabase
3. Vous arrivez au tableau de bord

## 🎯 Fonctionnalités

### Aperçu (Dashboard)

Affiche :
- **Total Produits** : Nombre de produits en stock
- **En Stock** : Produits avec stock > 0
- **Ruptures** : Produits à 0 stock
- **En Promotion** : Produits avec réduction

### ➕ Ajouter un produit

1. Cliquez sur **"Ajouter un produit"**
2. Remplissez les champs (tous les champs ont une grande taille pour mobiles)
3. Uploadez une photo
4. Cliquez **"Publier"**
5. ✅ Le produit apparaît immédiatement sur le site

### ✏️ Modifier un produit

1. Sélectionnez un produit dans la liste
2. Modifiez les informations
3. Cliquez **"Enregistrer"**

### 🗑️ Supprimer un produit

1. Cliquez sur l'icône poubelle
2. Confirmez la suppression
3. ✅ Le produit est supprimé

## 💡 Conseils

- **Photos** : Utilisez des images claires et bien éclairées
- **Prix** : Entrez le prix en FCFA (ex: 15000)
- **Tailles** : Cochez les tailles disponibles
- **Stock** : Mettez à jour régulièrement après chaque commande
- **Promo** : Entrez le pourcentage de réduction

## 📱 Usage sur téléphone

Le tableau de bord est complètement responsive :
- Grands boutons faciles à toucher
- Champs de saisie grands et clairs
- Navigation simple
- Tous les éléments visibles sans scroll horizontal

## ⚡ Trucs rapides

- Ajouter un produit en moins de 1 minute
- Modifier directement depuis le téléphone
- Voir immédiatement les changements sur le site
- Gérer les stocks en temps réel
